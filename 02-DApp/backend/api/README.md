# API

## Installation

```
npm install
```

## Usage

```
node app.js
```

## Endpoints

- (GET) /allAthletes/:week --> This will return all stats of all athletes for any given week.
- (GET) /athlete/:name --> Returns all of an athletes stats for each week
- (PUT) /athleteData --> Pushes the athlete data to DB
